## 1. Die Story

### Kernnarrativ
**"Berlin braucht seine Fußball-Woche zurück."**

102 Jahre Berliner Fußball-Geschichte. Letzte Ausgabe Oktober 2025. Insolvenz. Und jetzt: Ein Neustart — Digital First, getragen von der Community. Nicht von einem Medienkonzern, sondern von den Menschen, die jeden Samstag am Spielfeldrand stehen.

### Claim-Vorschläge (zum Diskutieren)
- **"Wieder da. Für euch."**
- **"102 Jahre Pause? 6 Monate."**
- **"Von der Kreisliga bis zur Regionalliga — Berlin erzählt wieder."**
- **"Die Fußball-Woche lebt."**

### Absender
- **Tino Loest** — Vorsitzender Eintracht Mahlsdorf, hat die Marke aus der Insolvenz gerettet
- **Redaktion** — wird aufgebaut (Chefredakteure + freie Mitarbeiter)
- **Community** — die eigentlichen Retter (→ Crowdfunding)

## 2. Phasen-Fahrplan

| Phase | Zeitraum | Fokus | Ziel |
|-------|----------|-------|------|
| **Phase 0: Seed** | Jetzt – Ende März | Teaser, Community aufwärmen, E-Mail-Liste | 500+ Newsletter-Anmeldungen |

## 3. Crowdfunding-Strategie

### Plattform
Startnext (DE-fokussiert, Fußball-Community kennt es) oder Kickstarter/GoFundMe (internationale Reichweite weniger relevant). **Empfehlung: Startnext.**

### Funding-Ziel
Festlegen: Was braucht die FuWo für die ersten 6 Monate?
- Redaktion (Chefredakteure, Freelancer)
- Druck erste Ausgabe
- Tech/Hosting
- → Realistisches Ziel: **25.000–50.000€**

### Reward-Stufen (Entwurf)

| Betrag | Reward | Name |
|--------|--------|------|
| **5€** | Danke-Nennung auf der Website (Supporter-Wall) | **Kreisliga-Supporter** |
| **15€** | + Digitaler FuWo-Sticker-Pack + Name in der ersten Ausgabe (Kleingedrucktes) | **Bezirksliga-Supporter** |
| **30€** | + Erste Print-Ausgabe nach Hause + exklusives Poster (Retro-FuWo-Cover) | **Berlin-Liga-Supporter** |
| **50€** | + FuWo-Schal oder -Mütze (Merch) + 3 Ausgaben | **Oberliga-Supporter** |
| **100€** | + Jahresabo Print + Name prominent auf der Website | **Regionalliga-Supporter** |
| **250€** | + Einladung zum Launch-Event + signierte Erstausgabe | **Bundesliga-Supporter** |
| **500€** | + 1/4-Seiten-Anzeige in der ersten Ausgabe (für Verein, Business, etc.) | **Ehrentribüne** |
| **1.000€** | + Namentliche Erwähnung in der Zeitung (eigene Rubrik "Gründungsunterstützer") + volle Seite Danksagung | **Gründungsmitglied** |
| **2.500€+** | + Persönliches Gespräch mit Redaktion + dauerhaftes Logo als Gründungspartner auf der Website | **Gründungspartner** |

> Liga-Naming der Stufen spielt die FuWo-DNA: Jeder kennt die Hierarchie.

### Auf der Website
- **Eigene Crowdfunding-Landingpage** (`/unterstuetzen` oder `/crowdfunding`)
- Fortschrittsbalken (Funding-Stand in Echtzeit)
- Supporter-Wall (Namen aller Unterstützer)
- Countdown bis Kampagnenende
- Direkte Einbindung auf der Startseite (Hero-Banner während der Kampagne)
- CTA in jeder Seite (sticky Footer-Bar: "Unterstütze die FuWo")

## 4. Social Media Strategie

### Plattformen & Priorität

| Plattform | Rolle | Frequenz |
|-----------|-------|----------|
| **Instagram** | Hauptkanal, Community, Storytelling | Täglich (Story) + 4-5 Posts/Woche |
| **TikTok** | Reichweite, junge Zielgruppe, Viralität | 2-3 Videos/Woche (Repurpose von IG Reels) |
| **Facebook** | Ältere Zielgruppe, Vereins-Gruppen, Events | 3 Posts/Woche (Cross-Post von IG) |
| **Newsletter** | Direkter Kanal, Conversion, Loyalität | 1x/Woche ("Kurzpass") |

### Content-Formate (basierend auf eurer Priorisierung)

**Prio 1: Meta Storytelling (Video/TT first)**
- "Die FuWo ist zurück" — Ankündigungs-Reel
- "Warum Berlin seine Fußball-Woche braucht" — Tino erzählt die Geschichte
- "Tour de Ring: Berliner Vereine" — Stadion-Besuche, Vereinsportraits
- Behind-the-Scenes: Redaktion aufbauen, erste Ausgabe planen

**Prio 2: Spieler:innen & Personen (HQ Bilder, Posts)**
- Gesichter des Berliner Amateurfußballs
- "Wer bist du?" — Kurzinterviews mit Spieler:innen, Trainer:innen, Ehrenamt
- Retro: Legenden der FuWo (Archiv-Fotos!)

**Prio 3: Vibe (Fun, Classics, Relatable)**
- Kabinen-Momente, Platztage, Derbyatmosphäre
- Nostalgische FuWo-Cover aus dem Archiv
- Memes & relatables für die Amateur-Community

**Prio 4: Stats (Informativer Content)**
- Tabellen-Updates, Spieltags-Zusammenfassungen
- "Wusstest du?" — Berliner Fußball-Fakten
- Infografiken: Vereinslandkarte Berlin

**Prio 5: Identität (Ausgabe im Fokus)**
- Reminder an die Tradition
- Cover-Reveals (neue und historische)
- "Letzte Woche in der FuWo" — Wochenrückblick

### Content-Stränge für die Crowdfunding-Phase

| Strang | Beschreibung | Frequenz | Ziel |
|--------|-------------|----------|------|
| **Countdown** | "Noch X Tage bis zum Launch" | Täglich (Story) | Spannung aufbauen |
| **Origin Story** | Wie die FuWo gerettet wurde | 3 Teile (Reel) | Emotionale Bindung |
| **Community Voices** | Vereinsvertreter, Fans sagen warum sie die FuWo brauchen | 2x/Woche | Social Proof |
| **Behind the Scenes** | Redaktion, Website, erste Artikel | 1-2x/Woche | Transparenz |
| **Supporter Shoutouts** | "Danke an [Name] — unser neuester Regionalliga-Supporter!" | Bei jeder Spende | FOMO + Dankbarkeit |
| **Milestone Celebrations** | "10%, 25%, 50% erreicht!" | Bei Milestone | Momentum |
| **Heritage** | Archiv-Cover, historische Momente | 2x/Woche | Identität stärken |

## 6. Newsletter-Strategie ("Kurzpass")

### Phase 0 (Jetzt)
- **Ziel:** E-Mail-Liste aufbauen
- **Content:** "Wir kommen zurück" — Teaser, Behind-the-Scenes, Countdown
- **Frequenz:** Alle 2 Wochen
- **CTA:** Auf Crowdfunding vorbereiten

### Phase 1 (Crowdfunding)
- **Spezial-Newsletter** am Kampagnenstart
- **Wöchentliche Updates:** Funding-Stand, neue Rewards, Supporter-Highlights
- **CTA:** Spenden + Teilen

### Phase 2 (Digital Launch)
- **Wöchentlicher "Kurzpass":** Top 5 Artikel der Woche, Tabellen-Updates, ein Kommentar
- **Personalisiert:** nach Lieblings-Liga (Onboarding-Daten)

## 7. PR-Strategie

### Die Geschichte für die Presse
**Headline:** "102 Jahre Tradition: Die Fußball-Woche kehrt zurück — Digital First, getragen von der Community"

### Key Messages
1. Letzte Berliner Sportzeitung wurde gerettet — von Vereinsmenschen, nicht von Investoren
2. Crowdfunding: Berlin finanziert seine eigene Zeitung
3. Digital First: Moderne Plattform für Amateurfußball
4. Inklusion: Frauen, Jugend, Senioren — nicht nur Herren-Oberklasse

### Ziel-Medien

| Medium | Winkel | Kontakt |
|--------|--------|---------|
| **rbb24 / Abendschau** | Lokale Comeback-Story | Sportredaktion |
| **Berliner Morgenpost** | Tradition + Innovation | Sportressort |
| **Tagesspiegel** | Community-Journalismus | Medien/Sport |
| **BZ** | "Kult-Zeitung kehrt zurück" | Sport |
| **kicker** | Fachpresse, Amateurfußball-Story | Redaktion |
| **11 Freunde** | Fußballkultur, Nostalgie | Redaktion |
| **Berliner Zeitung** | Startup/Medien-Winkel | Wirtschaft/Medien |
| **BFV (Berliner Fußball-Verband)** | Offizielle Unterstützung, Multiplikator | Pressestelle |
| **DFB.de** | Amateurfußball-Förderung | Amateur-Redaktion |

### PR-Timeline

| Wann | Was |
|------|-----|
| **2 Wochen vor Crowdfunding** | Exklusiv-Story an rbb oder Morgenpost pitchen |
| **Kampagnenstart** | Pressemitteilung an alle + Social Blitz |
| **Bei 50% Funding** | Follow-up: "Berlin steht hinter der FuWo" |
| **Kampagnenende** | Ergebnis-Meldung + Ausblick auf Launch |
| **Digital Launch** | Launch-Event + Presseeinladung |

### Wer muss dabei sein?
- **Tino Loest** — Gesicht der Rettung, Glaubwürdigkeit
- **BFV-Vertreter** — offizieller Segen
- **Vereinsvertreter** (2-3 Berliner Vereinspräsidenten) — "Wir brauchen die FuWo"
- **Alte FuWo-Redakteure** (wenn möglich) — Brücke zur Tradition
- **Prominente Berliner Fußball-Gesichter** — Botschafter (z.B. Ex-Hertha/Union-Spieler mit Amateur-Bezug)
